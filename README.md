## Overview
This project creates a simple blog site.
## Tech Stack
Backend: FastAPI <br>
Templates: Jinja2 (built into FastAPI via Starlette)<br>
Database: PostgreSQL<br>
ORM: SQLAlchemy<br>
Server: Uvicorn<br>
Styling: Simple CSS <br>
## Features
1. Create a post
2. View a post
3. Edit a post
4. Delete a post
